0ms 瞬间起跳弹出

按下快捷键瞬间 0ms 闪电弹出，异步加载及语法高亮，主线程永远保持 100% 流畅，不卡 Finder 也不丢帧。

Office 办公原生预览

Word, Excel, PPT, PDF, CSV 以及 iWork 格式无损展现。12px 物理圆角无缝贴合容器，消除直角分层与投影阴影干扰。

Markdown 一键导出 PDF

内置 PDF 编译器与 GitHub CSS。一键调起系统的 NSSavePanel 另存为 PDF，路径智能对齐，配备优雅的线性进度条。

原生 matchedGeometry 飞回

高精度探测 Finder 被选文件图标的物理屏幕坐标。关闭窗口时伴随 CASpringAnimation 精准飞回文件图标原位。
